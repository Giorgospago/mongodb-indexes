# Export and Import MongoDB Indexes from one database to another

## Install
### Step 1
```shell
git clone git@github.com:Giorgospago/mongodb-indexes.git
cd mongodb-indexes
npm i
cp .env.example .env
```
### Step 2
Fill your export and import uris at your `.env` file
### Step 3
```shell
npm start
```

# That's All !!!
